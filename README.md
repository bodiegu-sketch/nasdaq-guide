# 纳指介绍 - GitHub Pages 静态版

这是一个可直接部署到 GitHub Pages 的静态网站版本。

## 文件

- `index.html`: 页面入口
- `style.css`: 页面样式
- `og.png`: 分享预览图
- `favicon.svg`: 网站图标

## 发布到 GitHub Pages

这个仓库包含 GitHub Actions Pages 发布配置。推送到 `main` 后，可在仓库 `Settings` -> `Pages` 中选择 GitHub Actions 作为发布来源。

最终地址通常是：

```text
https://bodiegu-sketch.github.io/nasdaq-guide/
```
